### Why w3docs?

- **Explore any contract** — read functions, write calls, events, all in one page.
- **Try it live** — call functions directly from the docs, no terminal needed.
- **Multi‑chain by design** — Ethereum, Solana, Starknet, Sui, and more to come.
- **Zero setup** — just the contract address. No compilers, no source files.
- **Always up to date** — docs reflect the live contract, not a stale snapshot.

### How it works

Run `w3docs`, answer a few quick prompts, and your documentation is ready.
Share it as a single page, embed it anywhere, or publish it for the world.

### Built for every chain

The same clean, consistent experience across multiple chains. New chains are added regularly as the ecosystem grows.
